# ⛳ 골프 캘린더

골프 라운드 일정을 관리하는 모바일 최적화 웹 애플리케이션입니다.

## 주요 기능

### 📅 캘린더 & 일정 관리
- 월별 캘린더 뷰에서 일정 확인 및 날짜 선택
- 일정 상태별 색상 구분 (🟢 확정 / 🟡 미정 / 🟣 예정)
- 일정 등록/수정/삭제, 복수 골프장 선택
- 티오프 시간, 코스명, 동반자, 메모 기록
- 지난 일정 숨기기 옵션

### 💰 비용 관리
- 그린피, 카트비, 캐디피, 식사/기타 비용 기록
- **천 단위 콤마 자동 포맷** 입력
- **직전 일정의 비용 자동 채움** — 골프장 선택 시 해당 골프장의 직전 비용으로 덮어쓰기
- 카트비/캐디피 10만원 이상 시 인원수로 자동 분할 계산
- 총액/평균/최고 비용 통계, 월별·골프장별 비용 분석

### 👥 동반자 관리
- 동반자별 라운드 횟수 랭킹
- 등록된 사용자 동반자에게 일정 자동 공유
- 초대 메시지 공유 (Web Share API / 클립보드)
  - 그린피 40만원 이상 시 인원수로 분할 표시

### 📊 분석 (통계 / 동반자 / 비용 통합 탭)
- 전체 라운드 수, 상태별 현황, 자주 가는 골프장 순위
- 동반자 랭킹
- 월별·골프장별 비용 추이

### 🔐 사용자 관리
- 이메일 회원가입/로그인 (8자 이상, 영문+숫자 필수)
- 비밀번호 찾기 (이메일 발송)
- 관리자 패널 (사용자 조회, 권한 변경, 삭제)
- 마지막 로그인 시간 기록

### ⛳ 골프장 관리
- 기본 골프장: 아시아나, 렉스필드, 태광, 설해원
- 커스텀 골프장 추가/삭제

### 🔔 알림
- 다가오는 라운드 D-Day 알림 (8일 전부터)
- 예약 취소 확인 배너
- 브라우저 푸시 알림 지원

## UX / 디자인 시스템

- **2026 트렌드** Aurora 메쉬 그라데이션, Glass morphism, Floating FAB 내비게이션
- **3-탭 구조**: 캘린더 / 목록 / 분석(통계·동반자·비용 세그먼트)
- **접근성**: WCAG AA 대비, 44px+ 터치 타깃, `prefers-reduced-motion`, 키보드 포커스 링, ARIA 라벨링
- **모바일 최적화**: Safe-area inset, 한국어 폰트 폴백 (Noto Sans KR → Apple SD Gothic Neo)
