# ⛳ 골프 캘린더

골프 라운드 일정을 관리하는 모바일 최적화 웹 애플리케이션입니다.

## 주요 기능

### 📅 캘린더 & 일정 관리
- 월별 캘린더 뷰에서 일정 확인
- 일정 상태별 색상 구분 (🟢 확정 / 🟡 미정 / 🟣 예정)
- 일정 등록/수정/삭제
- 티오프 시간, 골프장, 동반자, 비용 기록

### 💰 비용 관리
- 그린피, 카트, 캐디피, 기타 비용 기록
- 총액/평균/최고 비용 통계
- 월별, 골프장별 비용 분석

### 👥 동반자 관리
- 동반자별 라운드 횟수 통계
- 동반자 랭킹

### 📊 통계
- 전체 라운드 수, 상태별 현황
- 자주 가는 골프장 순위
- 비용 추이 분석

### 🔐 사용자 관리
- 이메일 회원가입/로그인
- 비밀번호 찾기 (이메일 발송)
- 관리자 패널 (사용자 조회, 권한 변경, 삭제)
- 마지막 로그인 시간 기록

### ⛳ 골프장 관리
- 기본 골프장: 아시아나, 렉스필드, 태광, 설해원
- 커스텀 골프장 추가/삭제

### 🔔 알림
- 다가오는 라운드 D-Day 알림
- 브라우저 푸시 알림 지원
